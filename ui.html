Benjis.UI Framework

A lightweight, theme-driven UI framework for Garry’s Mod.

---

Theme

Benjis.UI.Theme


Benjis.UI.Theme = {
    BG      = Color(18,18,18,235),
    Panel   = Color(25,25,25,220),
    Hover   = Color(35,35,35,220),
    Accent  = Color(200,60,60),
    Divider = Color(0,0,0,180),
    Text    = color_white,
    Muted   = Color(160,160,160)
}

Defines global colors used by all framework components.

---

Windows

CreateFrame

Benjis.UI:CreateFrame(title, width, height, style)


local frame = Benjis.UI:CreateFrame(
    "My Window",
    600,
    400,
    "center"
)

Parameters:

  • title (string) – Window title
  • width (number) – Frame width
  • height (number) – Frame height
  • style (string) – "sidebar" or "center"
Sidebar windows slide in from the left.
Center windows fade + slide vertically.
---

Panels

CreatePanel

Benjis.UI:CreatePanel(parent, style)


local pnl = Benjis.UI:CreatePanel(frame.Content, "panel")
pnl:Dock(FILL)

Styles:

  • panel – Default content panel
  • bg – Background panel
  • transparent – Layout only
  • card – Rounded card style
---

Buttons

CreateButton

Benjis.UI:CreateButton(parent, text, height)


local btn = Benjis.UI:CreateButton(panel, "Click Me", 42)
btn:Dock(TOP)

btn.DoClick = function()
    print("Clicked")
end

Buttons automatically handle hover animation and theming.

---

Scroll Panels

CreateScroll

Benjis.UI:CreateScroll(parent)


local scroll = Benjis.UI:CreateScroll(panel)
scroll:Dock(FILL)

Creates a themed scroll panel with a minimal accent scrollbar.

---

Lists

CreateList

Benjis.UI:CreateList(parent)


local list = Benjis.UI:CreateList(panel)

AddListRow

Benjis.UI:AddListRow(list, text, stripColor)


Benjis.UI:AddListRow(list, "Player Name", team.GetColor(ply:Team()))

Rows are clickable buttons with optional color strip.

Animation & Settings


Benjis.UI.AnimTime = 0.25
Benjis.UI.Padding  = 12

Global animation timing and layout spacing.

---

Best Practices

  • Do not paint raw DPanels outside the framework
  • Use semantic components (Panel, Button, List)
  • Let the theme control visuals
  • Keep addon files focused on layout & logic
This framework is designed to scale across F4 menus, scoreboards, admin panels, and modal windows.
