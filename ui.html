Benjis RP UI Framework

A lightweight (hopefully), Performance based (hopefully again) , consistent UI Framework for Benji's Downtown & Dev servers.

Overview

Benjis.UI provides reusable UI components built on top of GMod VGUI:

  • Themed windows (sidebar & center)
  • Panels and layout containers
  • Animated buttons
  • Scroll panels and lists
  • Model panels
  • Responsive font system with presets
  • Centralized theme configuration

Theme System (cl_theme.lua)

Benjis.UI.Theme = {
    BG        = Color(18,18,18,235),
    Panel     = Color(25,25,25,220),
    Hover     = Color(35,35,35,220),
    Accent    = Color(200,60,60),
    Divider   = Color(0,0,0,180),
    Text      = color_white,
    Muted     = Color(160,160,160)
}

Benjis.UI.AnimTime = 0.25
Benjis.UI.Padding  = 12

Windows & Frames (cl_window.lua)

Create Frame

local frame = Benjis.UI:CreateFrame(title, width, height, style)
ParameterDescription
titleWindow title text
widthFrame width
heightFrame height
style"sidebar" or "center"

Styles

Sidebar — slides in from the left and slides out when closed.

Center — fades and slides vertically from the center.

Content Panel

frame.Content

Use this as the parent container for all UI elements inside the frame.

Panels (cl_panel.lua)

local pnl = Benjis.UI:CreatePanel(parent, style)
StyleDescription
panelDefault solid panel
bgBackground panel
transparentLayout-only container
cardRounded card with outline

Buttons (cl_button.lua)

Color Lerp Helper

function Benjis.UI:LerpColor(t, from, to)
    return Color(
        Lerp(t, from.r, to.r),
        Lerp(t, from.g, to.g),
        Lerp(t, from.b, to.b),
        Lerp(t, (from.a or 255), (to.a or 255))
    )
end

Create Button

local btn = Benjis.UI:CreateButton(parent, text, height)
  • Smooth hover animation
  • Theme-aware coloring
  • Font preset usage
  • Fully custom painted

Fonts (cl_fonts.lua)

Low-level Font Getter

Benjis.UI:GetFont(name, size, options)

Fonts automatically scale based on screen resolution and are cached for performance.

Font Presets

Benjis.UI:RegisterFontPreset("Button", {
    font = "Tahoma",
    size = 16,
    options = { weight = 600 }
})

Available presets:

  • Title
  • Header
  • Text
  • Small
  • Button
  • Mono

Scroll Panels (cl_scroll.lua)

local scroll = Benjis.UI:CreateScroll(parent)

Creates a themed DScrollPanel with a slim accent-colored scrollbar.

Lists (cl_list.lua)

local list = Benjis.UI:CreateList(parent)
local row = Benjis.UI:AddListRow(list, text, stripColor)

List rows are clickable buttons and may optionally display a color strip (e.g. team color).

Model Panels (cl_model.lua)

local mdl = Benjis.UI:CreateModelPanel(parent, modelPath)

Creates a static DModelPanel with preset camera and lighting values.

Best Practices

  • Do not paint raw VGUI elements outside the framework
  • Use font presets instead of hardcoded fonts
  • Let the theme control visual appearance
  • Keep addon logic separate from UI styling
